# 花蓮有機農地智慧轉型平台 - 農民端 Dashboard

## 📋 專案概述

這是一個專為花蓮地區農民設計的智慧農業 Dashboard，提供即時農地監控數據和轉型效益預估。採用自然綠色與土地棕色的配色方案，呈現永續農業的專業形象。支援完整的響應式設計和互動式 modal 系統。

## ✨ 主要功能

### 📊 農地總覽卡片區塊
- **土壤濕度監控**：即時顯示土壤濕度百分比，點擊查看詳細趨勢圖
- **氣溫監控**：當前農地氣溫數據，支援7天歷史記錄
- **光照監控**：光照強度 (lux) 即時數據，包含效率分析
- **二氧化碳濃度**：空氣品質監控 (ppm)，環境影響評估

### 🪟 互動式 Modal 系統
- **詳細數據視圖**：點擊卡片開啟對應的詳細 modal
- **互動式圖表**：每個 modal 包含專屬的 Chart.js 圖表
- **智能狀態分析**：自動判斷數據狀態並提供建議
- **多種關閉方式**：點擊遮罩、關閉按鈕、ESC 鍵、手機滑動關閉

#### Modal 功能詳情
- **土壤濕度 Modal**：7天濕度趨勢圖、灌溉建議、狀態判斷
- **氣溫 Modal**：7天氣溫變化、日夜溫差警告、降溫建議
- **光照 Modal**：光照分布圖、效率分析、遮光建議
- **CO₂ Modal**：濃度趨勢圖、環境影響、有機農業效益

### 📈 氣溫變化折線圖
- 顯示最近7天的氣溫變化趨勢
- 使用 Chart.js 實現互動式圖表
- 響應式設計，支援各種螢幕尺寸
- 智能工具提示和數據標籤

## 🚀 快速開始

### 檔案結構
```
H-Change/
├── index.html              # 主要 HTML 文件
├── styles.css              # CSS 樣式文件
├── script.js               # JavaScript 功能文件
├── README.md               # 專案說明文件
└── favicon/                # 網站圖標資料夾
    ├── favicon.ico         # 傳統瀏覽器圖標
    ├── favicon-16x16.png   # 小尺寸圖標
    ├── favicon-32x32.png   # 標準尺寸圖標
    ├── apple-touch-icon.png # iOS 設備圖標
    ├── android-chrome-192x192.png # Android 圖標
    ├── android-chrome-512x512.png # 高解析度圖標
    └── site.webmanifest    # PWA 配置檔案
```

### 使用方式
1. 將所有檔案下載到同一個資料夾
2. 用瀏覽器開啟 `index.html` 檔案
3. 即可看到完整的農民端 Dashboard
4. 點擊卡片查看詳細數據
5. 手機上可滑動關閉 modal
